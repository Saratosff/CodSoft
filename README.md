## 📌 Projects Overview

### 🧮 Calculator (GUI)
A modern **Android-style calculator** built using Tkinter.

**✨ Features:**
- Clean dark UI
- Keyboard support
- Error handling
- Expression display

**▶ Run:**
```bash
python Calculator/Calculator.py
```

---

### 🔐 Password Generator (GUI)
A secure password generator with customizable options.

**✨ Features:**
- Adjustable length (8–64)
- Toggle character types
- Secure generation using `secrets`
- Copy to clipboard

**▶ Run:**
```bash
python Password_generator/password_generator.py
```

---

### 🎮 Rock Paper Scissors (GUI Game)
An advanced animated version of the classic game.

**✨ Features:**
- Interactive animations
- Score tracking
- Match system (3, 5, 7 rounds)
- Game statistics

**▶ Run:**
```bash
python Rock_Paper_scissors/Rock_paper_scissors.py
```
